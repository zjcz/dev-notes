# Vi Text Editor

To open a file:

```bash
vi <filename>
```

While editing, press `i` to enter insert mode. Press `esc` to exit insert mode.

To save and exit:

```
:wq
```

or 

```
ZZ
```

To exit without saving:

```bash
:q!
```

To save and continue editing:

```bash
:w
```
